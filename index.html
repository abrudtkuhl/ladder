Ladder Chrome Extension

Ladder is a Chrome Extension that hoists you you over newspaper paywalls.

Currently hoists you over nytimes.com, washingtonpost.com, desmoinesregister.com

Note: Working on getting into the Chrome Web Store but for now you can install the extension manually. See Instuctions below...

Installation Instructions

  1. Right Click and Save extension file to your computer
  2. Click the Chrome Menu icon on the browser toolbar
  3. Select Tools > Extensions
  4. Locate the extension file on your computer and drag the file onto the Extensions page
  5. Review the list of permissions in the dialog that appears. If you would like to proceed, click Install

Release Notes

v0.5 - Stable Release Download Release
